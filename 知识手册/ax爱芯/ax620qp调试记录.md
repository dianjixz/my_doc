# AX620QP 开发
AX620QP 是爱芯元智面向行业市场推出的 3K 高清智能 IPC SoC。该芯片集成新一代智眸 4.0 AI-ISP，支持 4K@25fps 实时真黑光。同时集成新一代通元 4.0 高性能、场景优化的 NPU 引擎，使得 AX620QP 在高画质、智能处理和分析等方面领先行业水平。AX620QP 提供稳定，易用的 SDK 软件开发包，支撑客户产品快速量产。

AX620QP 的配置和 AX630C 很像，但是视频模块要比 AX630C 功能多一些，所以主大 IPC 设备芯片。
