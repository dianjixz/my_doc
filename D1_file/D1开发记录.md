# 2021.6.5号
今天移植了tslib的包，在移植该包的过程中出现了一些问题。    
主要问题，使用ts_calibrate时总是提示无法找到库。  
原因：`Configuration tslib features`里面没选择库，所以导致一直没找到模块。  
解决办法：只要把`Configuration tslib features`中的选喜爱嗯全部选中即可。
